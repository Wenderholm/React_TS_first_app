# Screenshots

W tym folderze możesz umieścić zrzuty ekranu aplikacji:

- `screenshot1.png` - Główny widok aplikacji
- `screenshot2.png` - Formularz dodawania celu
- `screenshot3.png` - Lista celów z przykładowymi danymi

## Jak zrobić zrzuty ekranu:

1. Uruchom aplikację (`npm run dev`)
2. Otwórz http://localhost:5173
3. Dodaj kilka przykładowych celów
4. Zrób screenshot i zapisz jako `screenshot1.png`
5. Zaktualizuj ścieżki w README.md jeśli potrzeba
